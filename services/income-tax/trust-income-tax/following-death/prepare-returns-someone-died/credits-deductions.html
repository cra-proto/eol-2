How to claim

All credits and deductions can be claimed on the personal income tax return 70(1).

However, if you are filing one or more optional returns, you might be able to:

  • you may be able to also claim a deduction or credit under the same line number on the optional return
    • in some cases, you can split deductions or tax credits between the returns
  • for lines 30000 to 30700 and 30900 to 31100, you can claim the full credit amount for each return that you file

Keep in mind that you can only file an optional return if you have eligible income to report on the return.

Find out if you have eligible income to report on the optional returns.

The T3 Trust Income Tax and Information Return has separate deductions and credits that can be claimed for the estate after the date of death.
