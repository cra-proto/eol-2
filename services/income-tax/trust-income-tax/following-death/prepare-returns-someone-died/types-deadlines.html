Filing deadlines

Deadlines might differ for the different types of returns you need to file for someone who died. If returns are filed late and there is a balance owing, the CRA will charge a late-filing penalty.

Deadlines by type of return

Final T1 income tax return 70(1)

Deadline:

The later of:

  • April 30 of the year following the death (the date of death falls between January and October)
  • 6 months following the death on the same calendar day as the death (the date of death falls during November or December)
Extended deadline if self-employed (running a business)

If the person who died or their spouse or common-law partner was running a business on the date of death, your deadline for the final return is extended to the later of:

  • June 15 of the year following the death (the date of death falls between January 1 and December 16)
  • 6 months following the death on the same calendar day as the death (the date of death falls between December 17 and December 31)

You will not be considered eligible for this extended due date if the business's expenditures are mainly for cost or capital cost of tax shelter investments.

Carrying a loss back from the T3 Trust Income Tax and Information Return

If you are amending your final return after making an election in a T3 return to carry back losses to the final return, you will have to file the amended final return by the later of:

  • the filing due date of the deceased person’s final T1 return that the legal representative is required to file or has elected to file
  • the filing due date for the estate’s T3 return for its first tax year

In certain situations, the CRA may allow you to file a late election, including the amended T1. Late, amended, or revoked elections.

For more information, refer to Capital losses in the T3 can be carried back to the final personal return.

File the return early

You are able to file the return during the year of the death.

Optional T1 returns for someone who died

Find out if you have eligible income to report on the optional returns.

Deadlines:

Return for rights or things 70(2)
  • 90 days after the mailing date of the Notice of Assessment or Reassessment for the final return.
Return for a partner or proprietor 150(4)
  • You must file this by the same deadlines as the final personal income tax return, generally April 30.
Return for a Graduated Rate Estate (GRE) 104(23)(d)
  • You must file this by the same deadlines as the final personal income tax return, generally April 30.

Income tax return for the year preceding the year of death

If you need to file the previous year return for someone who died between December 31 and the filing deadline for the previous year, the deadline for the previous year return is:

  • 6 months following the date of death

Find out if you need to file a previous year return for someone who died.

T3 Trust Income Tax and Information Return

Deadline:

  • 90 days after the end of the trust's tax year or wound-up date - the end of a Graduated Rate Estate's tax year should be decided by the filer and indicated in the first T3 return for the GRE. The end of the GRE's tax year can be any date up to 1 year from the date of death. A GRE can only continue for up to 3 years. Other types of trusts generally have a tax year end of December 31st.

File returns early

All the returns can be filed during the year of death, with the tax return from the previous year.

If you choose to file early, write the year at the top of the latest T1 or T3 return available. It will be assessed according to the lastest year's tax rules. You can request a reassessment the following year.

Example of filing a return early

If a person dies early in 2022, the legal representative may choose to file the final return in 2022. The return will generally be processed in that year. In these cases, the return will generally be processed using tax legislation applicable to the 2021 tax year. The legal representative can then request a reassessment of the return in the following year (2023) to apply any tax changes introduced for the 2022 tax year.

Refer to how to file a return using the previous year return package.

Penalties if you file a return late

If you file a return late and there is a balance owing, the CRA will charge a late-filing penalty.

The penalty is 5% of any balance owing, plus 1% of the balance owing for each full month that the return is late, to a maximum of 12 months. The late-filing penalty may be higher if the CRA charged a late-filing penalty on a return of the deceased for any of the three previous years. The CRA will also charge you interest on both the balance owing and any penalty.

Cancel or waive penalties or interest

In certain situations, the CRA may cancel penalties or interest if you file the return late because of circumstances beyond your control. Request to cancel or waive penalties or interest.
